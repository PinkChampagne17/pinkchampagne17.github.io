<h2 id="下载主题">下载主题</h2>
<h3 id="方式一直接下载源代码">方式一：直接下载源代码</h3>
<p>下载后放在 theme 文件夹里。</p>
<p><strong><a href="https://github.com/bolnh/hexo-theme-material/releases/tag/1.5.6" rel="noreferrer" target="_blank">Download latest release version</a></strong></p>
<h3 id="方式二使用-git-下载">方式二：使用 Git 下载</h3>
<p>你可以自己决定想要使用的分支；仅限开发者使用。</p>
<pre><code>cd themes
git clone https://github.com/viosey/hexo-theme-material.git material
cd material
git checkout {branch/tags name}
</code></pre>
<h2 id="应用主题">应用主题</h2>
<p>修改<strong><em>主目录</em></strong>的<code>_config.yml</code>文件中的<code>theme</code>属性为<code>material</code>。</p>
<pre><code class="language-YML">theme: material
</code></pre>
<h2 id="使用站内搜索插件">使用站内搜索插件</h2>
<p>安装 <a href="https://github.com/wzpan/hexo-generator-search" rel="noreferrer" target="_blank">hexo-generator-search</a> 插件。</p>
<p>修改<strong><em>theme/material</em></strong> 目录 <code>_config.yml</code> 中的 search 值</p>
<pre><code class="language-YML">search:
    use: local
    swiftype_key:
</code></pre>
<p>然后在<strong><em>主目录</em></strong>的<code>_config.yml</code>文件中添加</p>
<pre><code class="language-YML">search:
    path: search.xml
    field: all
</code></pre>
<h2 id="修改配置文件">修改配置文件</h2>
<p>修改<strong><em>theme/material/layout/_widget/dnsprefetch.ejs</em></strong>文件，否则会报错，修改内容如下：</p>
<pre><code>&#x3C;% } else if(theme.comment.use.startsWith("disqus")) { %>

// 修改为

&#x3C;% } else if(theme.comment.use &#x26;&#x26; theme.comment.use.startsWith("disqus")) { %>
</code></pre>
