<h2 id="示例系统">示例系统</h2>
<ul>
<li>Windows 10</li>
</ul>
<h2 id="安装-go-语言开发工具">安装 Go 语言开发工具</h2>
<p>根据自己的喜好在以下两个链接中，选择其中一个来下载 Go 语言开发工具，选择下载以<code>.msi</code>作为后缀名的 Windows 版。</p>
<ul>
<li>
<p><a href="https://studygolang.com/dl" rel="noreferrer" target="_blank">Go 语言中文网</a></p>
</li>
<li>
<p><a href="https://golang.org/dl/" rel="noreferrer" target="_blank">Go 语言官网</a>（需要魔法上网）</p>
</li>
</ul>
<h2 id="检查-go-语言开发工具是否可用">检查 Go 语言开发工具是否可用</h2>
<p>在命令行中直接输入<code>Go</code>命令，如果显示出如以下的提示，说明 Go 语言开发工具已经安装成功。</p>
<pre><code class="language-text">> go

Go is a tool for managing Go source code.

Usage:

        go &#x3C;command> [arguments]

The commands are:

        bug         start a bug report
        build       compile packages and dependencies
        clean       remove object files and cached files
        doc         show documentation for package or symbol
        env         print Go environment information
        fix         update packages to use new APIs
        fmt         gofmt (reformat) package sources
        generate    generate Go files by processing source
        get         add dependencies to current module and install them
        install     compile and install packages and dependencies
        list        list packages or modules
        mod         module maintenance
        run         compile and run Go program
        test        test packages
        tool        run specified go tool
        version     print Go version
        vet         report likely mistakes in packages
</code></pre>
<h2 id="安装编辑器">安装编辑器</h2>
<p>在这个示例中我们使用 VS Code 作为 Go 语言的编辑器。</p>
<h3 id="安装-vs-code">安装 VS Code</h3>
<ul>
<li><a href="https://code.visualstudio.com/" rel="noreferrer" target="_blank">VS Code 官网</a></li>
</ul>
<h3 id="安装-vs-code-插件">安装 VS Code 插件</h3>
<p>打开 VS Code，选择左侧的 Extensions，查找并安装以下两个用于<strong>中文化</strong>和<strong>Go 语言</strong>的插件。</p>
<ul>
<li>Chinese (Simplified) Language Pack for Visual Studio Code</li>
<li>Go</li>
</ul>
<h2 id="安装更新工具">安装/更新工具</h2>
<h3 id="配置代理">配置代理</h3>
<p>因为国内网络较为特殊的原因，需要配置代理才能通过网络安装/更新工具。</p>
<p>在命令行中输入以下命令配置代理。</p>
<pre><code class="language-text">> go env -w GO111MODULE=on
> go env -w GPROXY=https://goproxy.cn,direct
</code></pre>
<p>在 VS Code 中使用按<code>Ctrl</code>+<code>Shift</code>+<code>P</code>，在打开的输入框中输入<code>Go</code>，这时就可以看到名为<strong>Go: Install/Update Tools</strong>的选项。点击后将列出的所有工具勾上，再点击确认(OK)进行安装。</p>
<p>当 VS Code 自带的命令行出现类似以下的提示，安装项以<code>SUCCEEDED</code>结尾，说明该安装项已经安装成功。</p>
<pre><code class="language-text">Installing github.com/uudashr/gopkgs/v2/cmd/gopkgs SUCCEEDED
Installing github.com/ramya-rao-a/go-outline SUCCEEDED
Installing github.com/acroca/go-symbols SUCCEEDED
Installing golang.org/x/tools/cmd/guru SUCCEEDED
Installing golang.org/x/tools/cmd/gorename SUCCEEDED
Installing github.com/cweill/gotests/... SUCCEEDED
Installing github.com/fatih/gomodifytags SUCCEEDED
Installing github.com/josharian/impl SUCCEEDED
</code></pre>
<p>如果出现了类似以下提示，那就是有安装项安装失败了。</p>
<pre><code class="language-text">1 tools failed to install.
</code></pre>
<h2 id="hello-world">Hello, world!</h2>
<p>新建一个名为 hello-world 的文件夹，并在此文件夹下运行 go mod init {项目名}：</p>
<pre><code class="language-shell">> mkdir hello-world             # 新建名为hello-world文件夹
> cd ./hello-world              # 切换目录到该文件夹下
> go mod init hello-world       # 使用go mod进行初始化
</code></pre>
<p>新建一个名为<strong>main.go</strong>的文件，放入以下代码。</p>
<pre><code class="language-GO">package main

import "fmt"

func main() {
        fmt.Print("Hello, world!")
}
</code></pre>
<p>使用命令行<strong>进入 main.go 所在的目录</strong>，使用以下命令运行我们刚刚编写的 Hello world 程序。</p>
<pre><code class="language-text">> go run main.go
</code></pre>
<p>最后命令行打印出以下内容，说明刚刚编写的 Hello world 程序已经成功运行。</p>
