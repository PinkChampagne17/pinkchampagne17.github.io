<h2 id="示例系统">示例系统</h2>
<ul>
<li>Windows 10</li>
</ul>
<h2 id="安装-go-语言开发工具">安装 Go 语言开发工具</h2>
<p>根据自己的喜好在以下两个链接中，选择其中一个来下载 Go 语言开发工具，选择下载以<code>.msi</code>作为后缀名的 Windows 版。</p>
<ul>
<li>
<p><a href="https://studygolang.com/dl" rel="noreferrer" target="_blank">Go 语言中文网</a></p>
</li>
<li>
<p><a href="https://golang.org/dl/" rel="noreferrer" target="_blank">Go 语言官网</a>（需要魔法上网）</p>
</li>
</ul>
<h2 id="检查-go-语言开发工具是否可用">检查 Go 语言开发工具是否可用</h2>
<p>在命令行中直接输入<code>Go</code>命令，如果显示出如以下的提示，说明 Go 语言开发工具已经安装成功。</p>
<pre class="shiki shiki-themes vitesse-light vitesse-dark" style="background-color:#ffffff;--shiki-dark-bg:#121212;color:#393a34;--shiki-dark:#dbd7caee" tabindex="0"><code><span class="line"><span>> go</span></span>
<span class="line"><span></span></span>
<span class="line"><span>Go is a tool for managing Go source code.</span></span>
<span class="line"><span></span></span>
<span class="line"><span>Usage:</span></span>
<span class="line"><span></span></span>
<span class="line"><span>        go &#x3C;command> [arguments]</span></span>
<span class="line"><span></span></span>
<span class="line"><span>The commands are:</span></span>
<span class="line"><span></span></span>
<span class="line"><span>        bug         start a bug report</span></span>
<span class="line"><span>        build       compile packages and dependencies</span></span>
<span class="line"><span>        clean       remove object files and cached files</span></span>
<span class="line"><span>        doc         show documentation for package or symbol</span></span>
<span class="line"><span>        env         print Go environment information</span></span>
<span class="line"><span>        fix         update packages to use new APIs</span></span>
<span class="line"><span>        fmt         gofmt (reformat) package sources</span></span>
<span class="line"><span>        generate    generate Go files by processing source</span></span>
<span class="line"><span>        get         add dependencies to current module and install them</span></span>
<span class="line"><span>        install     compile and install packages and dependencies</span></span>
<span class="line"><span>        list        list packages or modules</span></span>
<span class="line"><span>        mod         module maintenance</span></span>
<span class="line"><span>        run         compile and run Go program</span></span>
<span class="line"><span>        test        test packages</span></span>
<span class="line"><span>        tool        run specified go tool</span></span>
<span class="line"><span>        version     print Go version</span></span>
<span class="line"><span>        vet         report likely mistakes in packages</span></span>
<span class="line"><span></span></span></code></pre>
<h2 id="安装编辑器">安装编辑器</h2>
<p>在这个示例中我们使用 VS Code 作为 Go 语言的编辑器。</p>
<h3 id="安装-vs-code">安装 VS Code</h3>
<ul>
<li><a href="https://code.visualstudio.com/" rel="noreferrer" target="_blank">VS Code 官网</a></li>
</ul>
<h3 id="安装-vs-code-插件">安装 VS Code 插件</h3>
<p>打开 VS Code，选择左侧的 Extensions，查找并安装以下两个用于<strong>中文化</strong>和<strong>Go 语言</strong>的插件。</p>
<ul>
<li>Chinese (Simplified) Language Pack for Visual Studio Code</li>
<li>Go</li>
</ul>
<h2 id="安装更新工具">安装/更新工具</h2>
<h3 id="配置代理">配置代理</h3>
<p>因为国内网络较为特殊的原因，需要配置代理才能通过网络安装/更新工具。</p>
<p>在命令行中输入以下命令配置代理。</p>
<pre class="shiki shiki-themes vitesse-light vitesse-dark" style="background-color:#ffffff;--shiki-dark-bg:#121212;color:#393a34;--shiki-dark:#dbd7caee" tabindex="0"><code><span class="line"><span>> go env -w GO111MODULE=on</span></span>
<span class="line"><span>> go env -w GPROXY=https://goproxy.cn,direct</span></span>
<span class="line"><span></span></span></code></pre>
<p>在 VS Code 中使用按<code>Ctrl</code>+<code>Shift</code>+<code>P</code>，在打开的输入框中输入<code>Go</code>，这时就可以看到名为<strong>Go: Install/Update Tools</strong>的选项。点击后将列出的所有工具勾上，再点击确认(OK)进行安装。</p>
<p>当 VS Code 自带的命令行出现类似以下的提示，安装项以<code>SUCCEEDED</code>结尾，说明该安装项已经安装成功。</p>
<pre class="shiki shiki-themes vitesse-light vitesse-dark" style="background-color:#ffffff;--shiki-dark-bg:#121212;color:#393a34;--shiki-dark:#dbd7caee" tabindex="0"><code><span class="line"><span>Installing github.com/uudashr/gopkgs/v2/cmd/gopkgs SUCCEEDED</span></span>
<span class="line"><span>Installing github.com/ramya-rao-a/go-outline SUCCEEDED</span></span>
<span class="line"><span>Installing github.com/acroca/go-symbols SUCCEEDED</span></span>
<span class="line"><span>Installing golang.org/x/tools/cmd/guru SUCCEEDED</span></span>
<span class="line"><span>Installing golang.org/x/tools/cmd/gorename SUCCEEDED</span></span>
<span class="line"><span>Installing github.com/cweill/gotests/... SUCCEEDED</span></span>
<span class="line"><span>Installing github.com/fatih/gomodifytags SUCCEEDED</span></span>
<span class="line"><span>Installing github.com/josharian/impl SUCCEEDED</span></span>
<span class="line"><span></span></span></code></pre>

<pre class="shiki shiki-themes vitesse-light vitesse-dark" style="background-color:#ffffff;--shiki-dark-bg:#121212;color:#393a34;--shiki-dark:#dbd7caee" tabindex="0"><code><span class="line"><span>1 tools failed to install.</span></span>

<pre class="shiki shiki-themes vitesse-light vitesse-dark" style="background-color:#ffffff;--shiki-dark-bg:#121212;color:#393a34;--shiki-dark:#dbd7caee" tabindex="0"><code><span class="line"><span style="color:#AB5959;--shiki-dark:#CB7676">></span><span style="color:#393A34;--shiki-dark:#DBD7CAEE"> mkdir hello-world             </span><span style="color:#A0ADA0;--shiki-dark:#758575DD"># 新建名为hello-world文件夹</span></span>

<span class="line"><span style="color:#AB5959;--shiki-dark:#CB7676">></span><span style="color:#393A34;--shiki-dark:#DBD7CAEE"> go mod init hello-world       </span><span style="color:#A0ADA0;--shiki-dark:#758575DD"># 使用go mod进行初始化</span></span>

<span class="line"><span style="color:#1E754F;--shiki-dark:#4D9375">func</span><span style="color:#59873A;--shiki-dark:#80A665"> main</span><span style="color:#999999;--shiki-dark:#666666">()</span><span style="color:#999999;--shiki-dark:#666666"> {</span></span>
