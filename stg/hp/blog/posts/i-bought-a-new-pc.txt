<p>在前段时间，经过我苦苦哀求，我的钱包终于同意让我购买一台新电脑。这台电脑有些与众不同，它采用了其他电脑通常不会使用的硬件，并且应该是目前所有官方支持 Linux 的电脑中最便宜的一款。</p>
<!-- more -->
<p>也许你已经猜到了，这台电脑它的其实是一个 AMD64 架构的掌机——<a href="https://store.steampowered.com/steamdeck" target="_blank" rel="noreferrer">Steam Deck</a>。</p>
<p>这里简单地介绍一下它的配置：</p>
<pre><code>AMD APU

CPU: Zen 2 4c/8t, 2.4-3.5GHz (up to 448 GFlops FP32)

GPU: 8 RDNA 2 CUs, 1.0-1.6GHz (up to 1.6 TFlops FP32)

APU power: 4-15W

16 GB LPDDR5 on-board RAM (5500 MT/s quad 32-bit channels)

7" 1280 x 800 (16:10) 60Hz IPS LCD

SteamOS 3.0 (Arch-based) KDE Plasma
</code></pre>
<p>它出厂内置了一个基于 Arch 的 SteamOS 系统，使用了 KDE 作为默认桌面环境，可以使用 pacman。但通常还是在一个类似 Steam 的大屏幕模式的 UI 上，去下载和运行游戏，不会直接去使用桌面环境。</p>
<p>我下载了 <del>原版原神</del> <a href="https://store.steampowered.com/app/601150" target="_blank" rel="noreferrer">鬼泣 5</a> 并简单地试玩了玩一下，感觉画质和流畅度表现都非常的不错。</p>
<p>但我其实更想使用这台设备玩 <del>吉卜力原神</del> <a href="https://store.steampowered.com/app/589360" target="_blank" rel="noreferrer">二之国：暗影国度</a>、 <del>回合制原神</del> <a href="https://store.epicgames.com/p/honkai-star-rail" target="_blank" rel="noreferrer">崩坏：星穹铁道</a> 以及 <del>万代原神</del> <a href="https://store.steampowered.com/app/775500" target="_blank" rel="noreferrer">绯红结系</a>。</p>
<p>我手上的绯红结系虽然是破解版，但是通过 Steam 的添加非 Steam 游戏功能并设置使用 Proton 运行，就可以正常游玩。</p>
<p>二之国虽然在兼容层的配合下可以运行，但是这个游戏没有官方中文支持，而且它的汉化补丁的是 exe 格式的。</p>
<p>星穹铁道因为反作弊系统不能简单地通过兼容层在 Linux 上运行，虽然可以通过一些方式解决（据说有封号风险），但我更偏好选择装一个 Windows 11 与内置的 SteamOS 组成双系统，这样以后在 SteamOS 不方便直接运行的游戏，我都可以切换到 Windows 上直接运行。</p>
<p>这台 Steam Deck 原本是 64GB 的版本，在到我手上之前，原先的硬盘被换为了西数 SN740 1TB 并使用官方提供的工具装上了原先预装的 SteamOS。</p>
<p>我使用 <a href="https://gparted.org/" target="_blank" rel="noreferrer">GParted</a> 的 Live 镜像将现有的分区中，分出了几百 G 并格式化为 NTFS 文件系统给 Windows 使用，最后成功安装上 Windows 11。不幸的是 Windows 11 不会为 Steam Deck 自动安装驱动，需要我去官网下载并手动安装。</p>
<p>真正让我头疼的是，我希望开机时还是以 SteamOS 作为默认系统启动，但<strong>每次</strong> Windows 11 启动后都会把它自己的 UEFI 启动顺序修改为最优先。<sub>（比尔·盖茨！萨提亚·纳德拉！）</sub></p>
<p>我使用 Steam 的<a href="https://help.steampowered.com/faqs/view/46BD-6BA8-B012-CE43" target="_blank" rel="noreferrer">本地网络游戏转移</a>功能，从我的旧电脑上下载已经安装好补丁的二之国，但最后看起来 Steam 还是顺便做了文件校验并重新下载了被修改的部分（应该是出于文件完整性与安全性的考量）。但最后二之国的汉化补丁我尝试在两个系统上都安装一遍，但奇怪的是实际进入游戏仍然不是中文。</p>
<p>很不幸，我想在这台设备上玩的游戏实际在这台设备上玩并不方便。或许我再花些时间就解决上面提到的这些问题，但我发现 Steam Deck 在实际到手之后，我并不会想着像买之前想的那样，躺在床上使用它来玩游戏。还是觉得使用显示器带来的效果对于我来说更有吸引力（特别是在我换了新显卡之后）。我也没有出门玩游戏的需求，更不会抛弃我的笔记本不用而是用它来做便携式生产力工具。</p>
